### 块状数组

给定一个数组[1,2,4,5], 应答两种询问
- [l:r] 区间和
- 将[l:r] 之间加入一个数

前缀和 1 O(1), 2. O(n)
差分数组 1 O(n), O(1)

线段树 O(logN)
块状数组 O(N^0.5) 
[[1,2],[2]] -> 整块散块
int[] sum  是第i块的个数和，
int[] lazy 是块的个数,当给这个整块所有元素加一个数

[l:r] -> O(2*sqrt N + sqrt N)
[l:r] 加入一个数 -> O(2*sqrt N + sqrt N)
